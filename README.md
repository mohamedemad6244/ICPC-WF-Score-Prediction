# ICPC WF Score Prediction

## Overview
This project predicts the performance scores of participants in ICPC (International Collegiate Programming Contest) using machine learning techniques. The analysis is implemented in a Jupyter Notebook and focuses on feature extraction, data preprocessing, and predictive modeling.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA) to understand trends
- Machine learning model training and evaluation
- Score prediction for participants based on historical data

## Technologies
- Python
- Jupyter Notebook
- Pandas, NumPy
- Scikit-Learn
- Matplotlib, Seaborn

## Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/mohamedemad6244/ICPC-WF-Score-Prediction.git

### Open the notebook:
1. jupyter notebook icpc-wf-score-prediction.ipynb
2. Follow the notebook cells to preprocess data, train models, and predict scores.
